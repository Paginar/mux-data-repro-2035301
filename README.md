## Get started

1. Install dependencies

   ```bash
   npm install
   or
   yarn install
   ```

2. Run prebuild

   ```bash
    npx expo prebuild --clean
   ```

3. Run the project on android

   ```bash
    yarn run android
    or
    npm run android
   ```

4. Or you can Run the project on ios

   ```bash
    yarn run ios
    or
    npm run ios
   ```
